## Installing

```shell
$ composer require /smallnews/laravel-shopcore -vvv
```

## publish

```
数据迁移
php artisan migrate

发布配置文件
php artisan vendor:publish --tag=shopcore-config

发布视图
php artisan vendor:publish --tag=shopcore-views

发布组件
php artisan vendor:publish --tag=shopcore-components
```
